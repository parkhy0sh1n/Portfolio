# FRONT-END SKILL

  • HTML5

    웹 접근성, 표준성을 준수하여 시멘틱 태그를 이용해 마크업 할 수 있습니다.

  • CSS3

    position, display를 적절히 사용하여 반응형 웹을 적용할 수 있습니다.

  • SASS

    SCSS의 문법과 모듈화를 잘 활용하여 스타일 작업을 할 수 있습니다.

  • JavaScript

    웹 페이지의 동적 기능을 구현할 수 있으며, 이벤트 처리, AJAX 요청 및 비동기 프로그래밍에 대한 이해가 있습니다.

  • JQuery

    웹 페이지에서 HTML 요소를 선택 후 조작하고 이벤트 처리, AJAX 요청, 애니메이션 적용 등을 할 수 있습니다.

  • Vue.js

    Vue 3을 사용하여 Vuex, Vue Router 등 활용 가능하며, 컴포넌트 기반 웹 애플리케이션을 개발할 수 있습니다.

  • React.js

    React 구조와 여러가지 hook들을 잘 알고 있으며, 범용적이고 재사용가능한 컴포넌트 설계를 할 수 있습니다.

  • Nexacro

    넥사크로 플랫폼을 다루어 엔터프라이즈급 어플리케이션 및 데이터 시각화를 효과적으로 구축할 수 있습니다.

  • Bootstrap

    컴포넌트 레이아웃을 빠르고 쉽게 만들기 위해 Bootstrap을 활용할 수 있습니다.

# BACK-END SKILL

  • Java

    자바 언어의 핵심 개념과 객체 지향에 대한 이해, 웹 개발, 데이터베이스 연동, 프레임워크 활용 등을 숙달하였습니다.

  • Spring

    스프링을 사용하여 웹 애플리케이션을 개발하고 실행하여 배포한 프로젝트 경험이 있습니다.

  • MyBatis

    SQL 쿼리와 자바 코드를 연결하고, 데이터베이스와 상호작용하는 방법을 이해하고 있습니다.

  • JSON

    JSON의 객체 구조를 잘 파악할 수 있으며, javascript에서 활용할 수 있습니다.

  • JWT

    JWT를 통해 안전하고 효율적인 인증 및 정보 교환을 할 수 있습니다.

  • Node.js

    서버의 API를 구축할 수 있으며, DB와 연동하여 CRUD 기능을 수행할 수 있습니다.

  • PHP

    PHP를 이용하여 동적컨텐츠를 만들 수 있고, CRUD 기능을 수행할 수 있습니다.
